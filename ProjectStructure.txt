# PHASE 5 — PAILLIER ENCRYPTION

## Goal

Build privacy-preserving voting.

## Features

* generate Paillier keys
* encrypt vote
* decrypt tally
* homomorphic addition

## Modules

```bash
backend/crypto/
├── paillier.js
├── keygen.js
├── tally.js
└── threshold.js
```

## Flow

```text
Vote Selection
→ Encrypt Vote
→ Submit Ciphertext
→ Aggregate Ciphertexts
→ Decrypt Final Result
```

# 5. CRITICAL SECURITY RULES

NEVER:

* store plaintext passwords
* store plaintext votes
* decrypt individual ballots
* allow duplicate tokens
* trust frontend validation

ALWAYS:

* bcrypt hash
* verify JWT
* validate election status
* verify token usage
* hash blockchain data
* verify signatures
